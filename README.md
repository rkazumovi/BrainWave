BrainWave 🧠

A fully responsive, fully-coded AI landing page kit built with React, Vite, and Tailwind CSS — based on the Brainwave UI8 design kit.

🔗 Live Demo: brain-wave-site.netlify.app
🎨 Design Source: Brainwave AI Landing Page Kit on UI8


Overview

BrainWave is a polished, modern landing page template designed for AI/SaaS products. It's a coded implementation of the original Brainwave Figma design kit, translated into a production-ready React application. The project provides a clean, customizable foundation for building marketing sites, product pages, and app landing pages with minimal setup.

The template includes premade sections for a full marketing site — hero, features, pricing, roadmap, and authentication pages — all built with reusable, composable components so you can mix, match, and rearrange sections to fit your own brand.

🚀 Getting Started

Prerequisites


Node.js (v16 or later recommended)
npm (comes with Node.js) or yarn


Installation


Clone the repository


bash   git clone https://github.com/rkazumovi/BrainWave.git
   cd BrainWave


Install dependencies


bash   npm install


Start the development server


bash   npm run dev


Open your browser and navigate to http://localhost:5173


Build for Production

bashnpm run build

The optimized production build will be output to the dist/ folder, ready to deploy to any static hosting provider (Netlify, Vercel, etc.).
